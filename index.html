bntools

tools for mega man battle network ロックマンエグゼのツール

welcome to my tools for mmbn. ロックマンエグゼ用ツールのホームページへようこそ。

  • n1gpedia: browse chips across all games n1gpedia： チップ情報
  • fullcust: navicust autolayout fullcust： ナビカスタマイザーパーツ自動配置
  • zsaver: inspect and edit save files zsaver： セーブデータ検査、編集
